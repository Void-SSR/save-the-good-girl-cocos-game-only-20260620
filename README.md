# 拯救美少女

微信小游戏版《拯救美少女》Cocos Creator 3.8.8 纯游戏工程。

## 项目定位

本项目是一个竖屏防线射击 + Boss 净化收集小游戏原型。玩家扮演勇者/净化者，抵御污染怪潮，击败并净化被神秘人污染成黑化 Boss 的美少女。被净化的 Boss 会成为同伴，继续帮助玩家拯救后续角色。

## 当前内容

- Cocos Creator 3.8.8 工程配置
- 微信小游戏基础配置
- 10 章关卡数据
- 10 名 Boss 人设基础数据
- 3 类敌人数据
- 5 个局内升级选项
- 战斗模型、存档、章节解锁、图鉴/编组逻辑骨架
- `MiniGameEntry.ts` 中的可视化原型入口

## 当前状态

这是一个可继续制作的游戏原型，不是完整成品。后续重点是补齐正式场景、UI Prefab、美术素材、Boss 技能、章节剧情、同伴助战和微信小游戏构建流程。
